Minimum Project Requirements

  • Client viewer application
  • User-selected movement & actions
  • Basic enemy AI using Alpha-Beta pruning
  • Path-finding using A*
  • Track experience to properly scale allies and enemies
  • Implement 4 allies, 4 enemies, and 1 ability per ally
  • Minimal delay (<1 second) between user and enemy moves
  • Implement (<1 minute) basic playable random map generation
